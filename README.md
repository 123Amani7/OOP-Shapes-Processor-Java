# **Project Overview**
A robust Java-based application designed to calculate and manage properties of geometric shapes. This project showcases core **Object-Oriented Programming (OOP)** principles such as Inheritance and Polymorphism.
### **Key Features**
* **Interactive GUI:** Built with Java Swing for real-time user interaction.
* **OOP Architecture:** Clean implementation of class inheritance and data encapsulation.
* **Automated Calculations:** Precise area and perimeter computation for circles, rectangles, and triangles.
